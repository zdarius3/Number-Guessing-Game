## Installation (C# 12 / .NET 8)
1. **Clone the repository:**

   ```bash
   git clone https://github.com/zdarius3/GitHub-User-Activity
   cd GitHub-User-Activity
   ```

2. **Build the project:**
    ```bash
    dotnet build
    ```
    

3. **Execute the program:**
    ```bash
    dotnet run
    ```
